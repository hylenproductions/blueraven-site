# Blue Raven

Open protocol and hardware marketplace for indie IoT builders.
Site: https://blueraven.build

## Stack
- Frontend: SvelteKit
- CMS: Sanity.io
- Commerce: Shopify Headless (Storefront API)
- Email: Resend
- Hosting: Vercel
- DNS: Cloudflare

## Setup
Copy .env.example to .env and fill in values before running.

## Dev
```
npm run dev
```
